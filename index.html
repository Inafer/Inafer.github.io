HTML

HTML (Hypertext Markup Language) adalah bahasa yang digunakan untuk membangun struktur dasar dari sebuah halaman web.

CSS

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mengatur tampilan dan desain halaman web.

JavaScript

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web menjadi interaktif.

Responsive Design

Responsive design adalah teknik untuk membuat halaman web dapat menyesuaikan tampilannya dengan berbagai perangkat, seperti komputer, tablet, dan smartphone.
